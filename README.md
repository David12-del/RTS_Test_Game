# Survival Game

Multiplayer 2D top-down survival game.

## Setup

```bash
npm install
npm start
```

Then open http://localhost:3000

## Controls

- WASD - movement
- 1-4 - select building
- Left click - place building
- Right click - shoot

## Buildings

1. Wall - 50 (blocks enemies)
2. Turret - 100 (auto shoots)
3. Dril - 150 (on resource points, gives 25)
4. Mine - 75 (gives 5)
